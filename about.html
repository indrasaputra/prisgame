About PRIS Game

PRIS Game is a game developed by Algorithm and Programming Laboratory, Department of Informatics, Faculty of Information Technology, Institut Teknologi Sepuluh Nopember.
This game is a modified version of a well-known problem in scope of algorithm, Knapsack Problem. To solve the game we used dynamic programming approach.
Just like knapsack problem, this game is a good way to learn dynamic programming. The problem isn't hard but you may get the main concept/sense of dynamic programming. The last, learning algorithm is fun :)
